* Solved
  * Exchange Argument
  * ..., X(a, b), Y(c, d), ...가 있을 때,
    * ..., X(a, b), Y(c, d), ...
    * ..., Y(c, d), X(a, b), ...
  * 위 두 경우 모두 ... 영역에 대한 값이 변경되지는 않는다.
    * X, Y의 순서 변경으로 인해 전체 시간만 추가됨
  * 따라서, 두 경우만 단순 비교하여 최적의 경우를 찾아낼 수 있음
    * t 시점을 두고 X(a, b), Y(c, d) 그리고 Y(c, d), X(a, b)로 구성하고
        비교하면,
    * b/a, d/c가 각각 나오고 둘 중 값이 작은 것을 먼저 수행하면 된다w:w
